      def Verso.Output.Html.mustClose :
      List String

      Tags for which tag omission rules do not permit omitting the closing tag (the XHTML-style <selfclosing/> syntax is also invalid here). This list includes all SVG tags, since SVG is XML based and XML requires that every tag be explicitly closed.
