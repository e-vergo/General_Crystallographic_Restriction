    structure SubVerso.Highlighting.InfoTable :
    Type

    A table that maps source regions to info.

    SubVerso has very different access patterns from the language server, and was needing to re-traverse the info tree in ways that took too much time while finding proof states. This solution pre-processes it once, saving the info in an easier-to-query form for this use case.

    Other info types may be added as needed for performance in the future.

    • tacticInfo : Compat.HashMap Compat.Syntax.Range (Array (Lean.Elab.ContextInfo × Lean.Elab.TacticInfo))
    • infoByExactPos : Compat.HashMap (String.Pos.Raw × String.Pos.Raw) (Array (Lean.Elab.ContextInfo × Lean.Elab.Info))

      All info nodes indexed by exact (start, end) position for O(1) lookup

    • termInfoByName : Compat.HashMap Lean.Name (Array (Lean.Elab.ContextInfo × Lean.Elab.TermInfo))

      TermInfo indexed by expression name (for const/fvar lookups)

    • nameSuffixIndex : Compat.HashMap String (Array Lean.Name)

      Index from final name component string to full names for O(1) suffix lookup

    • allInfoSorted : Array (String.Pos.Raw × String.Pos.Raw × Lean.Elab.ContextInfo × Lean.Elab.Info)

      All info nodes with positions, sorted by start position for containment queries. Each entry is (startPos, endPos, ctx, info). Sorted by startPos ascending.
