    Dressed Artifacts Serialization #

    This module provides the full dressed artifact format serialization. This is the primary format consumed by leanblueprint.

    Output Format #

    Per declaration:

    {
  "html": "<pre>...</pre>",
  "htmlBase64": "PHByZT4uLi48L3ByZT4=",
  "jsonBase64": "eyJoaWdobGlnaHRlZCI6Li4ufQ=="
}
