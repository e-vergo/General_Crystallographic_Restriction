          def Dress.splitAtDefinitionAssign (hl : SubVerso.Highlighting.Highlighted) (splitAtAssign : Bool := true) :
          SubVerso.Highlighting.Highlighted × Option SubVerso.Highlighting.Highlighted

          Split highlighted code at the definition's := token. This finds the := that separates the signature from the body by:

          1. Looking for def/theorem/lemma/abbrev keyword
          2. Finding the := at bracket depth 0 after that keyword

          If splitAtAssign is true (default): Returns (signature, body) where signature includes up to and including :=. If splitAtAssign is false: Returns (full code from def keyword, none) - strips prefix but doesn't split at :=.

          Always strips any prefix before the def keyword (e.g., @[blueprint ...]).
