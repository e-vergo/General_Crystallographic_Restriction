    HTML Rendering for Blueprint Highlighting #

    This module provides a thin wrapper around Verso's Highlighted.toHtml to render SubVerso highlighted code to HTML strings.

    Uses Genre.none and minimal configuration to avoid dependencies on Verso's full document infrastructure.
